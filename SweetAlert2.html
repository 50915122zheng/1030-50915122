SweetAlert2

sweetalert2官網

SweetAlert2 是一個美觀的彈出窗口工具，讓您可以輕鬆創建多種互動提示。

範例程式碼


            
            <link rel="stylesheet" href="style.css">
            <script src="https://cdn.jsdelivr.net/npm/sweetalert2@11"></script>

            // 基本提示

                Swal.fire('這是一個基本提示！');

            // 確認提示

                Swal.fire({
                    title: '您確定嗎？',
                    text: "此操作無法回復！",
                    icon: 'warning',
                    showCancelButton: true,
                    confirmButtonColor: '#3085d6',
                    cancelButtonColor: '#d33',
                    confirmButtonText: '是的，確定！'
                }).then((result) => {
                    if (result.isConfirmed) {
                        Swal.fire('已確認！', '您的操作已成功完成。', 'success');
                    }
                });

            // 輸入提示

                Swal.fire({
                    title: '請輸入您的名字',
                    input: 'text',
                    inputPlaceholder: '輸入名字',
                    showCancelButton: true
                }).then((result) => {
                    if (result.value) {
                        Swal.fire(`你好, ${result.value}!`);
                    }
                });
            //錯誤提示
                
                    Swal.fire({
                        icon: "error",
                        title: "錯誤",
                        text: "大錯特錯",
                        footer: 'Why do I have this issue?'
                    });
                
            //帶有自訂圖像的訊息
                
                    Swal.fire({
                        title: "香蕉!",
                        text: "猴子愛吃香蕉.",
                        imageUrl: "https://i.pinimg.com/736x/e2/4a/8c/e24a8c1d1613eeb8d9371ad84faf3cec.jpg",
                        imageWidth: 400,
                        imageHeight: 200,
                        imageAlt: "Custom image"
                    });
